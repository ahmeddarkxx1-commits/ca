# Sprint 02 — Design System

## Sprint Goal

Build the complete Design System for Nexoura Core.

This sprint does NOT implement business features.

It builds the reusable UI foundation that every future module will use.

---

## Objectives

Create a scalable enterprise UI system.

Support Light Mode and Dark Mode.

Support Arabic (RTL) and English (LTR).

Ensure all components are reusable.

Follow accessibility best practices.
